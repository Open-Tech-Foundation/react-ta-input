# Introduction

> The Thamizhl (தமிழ்) language scripts input for ReactJS.

## Features

✅ Simple API

✅ It is based on the [ta-input-spec](https://github.com/Open-Tech-Foundation/ta-input-spec) specification

✅ Virtual Keyboard

✅ Easy switch between `ta` & `System` input
